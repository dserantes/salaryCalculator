# 🧮 Calculadora Laboral España 2025

App web estática para calcular nómina, IRPF, finiquito, bonus y beneficios fiscales según la normativa española vigente en 2025.

🔗 **Demo:** [dserantes.github.io/salaryCalculator](https://dserantes.github.io/salaryCalculator)

---

## ✨ Funcionalidades

| Módulo | Descripción |
|---|---|
| **Nómina** | Bruto/neto mensual y anual con 12 o 14 pagas |
| **IRPF & SS** | Tramos 2025 (19%–47%), cotizaciones SS al 6,48% |
| **Jubilación** | Estimación de pensión según años cotizados y proyección salarial |
| **Finiquito** | Días pendientes, vacaciones, prorrata pagas extra e indemnización |
| **Bonus** | Cálculo neto con consecución de objetivo e IRPF marginal |
| **Ticket Restaurante** | Ahorro fiscal real (exento hasta 11€/día laborable) |
| **Multi-empresa** | % de retención óptimo para salir a 0 en la declaración (Modelo 145) |
